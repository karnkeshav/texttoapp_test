One-time setup

Ready in 5 minutes

Set up your GitHub workspace once — then build unlimited apps for free, forever.

Step 1

Download GitHub Desktop

The free app that manages your code files on your computer

  1. Go to desktop.github.com in your browser
  2. Click the purple "Download for [your OS]" button
  3. Open the downloaded file and install GitHub Desktop
  4. Launch GitHub Desktop — sign in to GitHub when prompted
  5. Click "Sign in to GitHub.com" and follow the steps
  6. Enter your name and email, then click "Finish"
💡 Don't have a GitHub account? Go to github.com and click "Sign up" — it's completely free.
Step 2

Create your project repository

A repository is like a folder for your website — GitHub will host it free

  1. Open GitHub Desktop → File → New Repository
  2. Give it a name using lowercase letters and dashes (e.g. my-portfolio)
  3. Tick "Initialize this repository with a README"
  4. Click "Create Repository"
  5. Click "Publish repository" → make sure it says Public
  6. Click "Publish Repository" — done!
  1. Open GitHub Desktop → File → Clone Repository
  2. Switch to the "GitHub.com" tab
  3. Click the repository you want to use, choose a save location
  4. Click "Clone"
💡 Remember your repository name — you'll select it in Ready4Launch when building.
Step 3

Enable GitHub Pages

This turns your repository into a live public website — free forever

  1. Go to github.com, open your repository
  2. Click "Settings" (gear icon) → "Pages" in the left sidebar
  3. Under "Branch", select "main" and leave folder as "/ (root)"
  4. Click "Save" — your site will be live in about 60 seconds
💡 Your site URL will be: yourusername.github.io/your-repo-name

Quick checklist & launch

Confirm everything is ready, then connect your GitHub
