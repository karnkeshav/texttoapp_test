Before you start

One-time setup guide

Follow these simple steps to prepare your workspace. You only do this once. Each step takes about 2 minutes.

Step 1

Download GitHub Desktop

The free app that manages your code files on your computer

  1. Go to desktop.github.com in your browser
  2. Click the purple "Download for [your OS]" button
  3. Open the downloaded file and install GitHub Desktop (just click Next/Install)
  4. Launch GitHub Desktop — it will ask you to sign in to GitHub
  5. Click "Sign in to GitHub.com" and follow the steps
  6. Enter your name and email when asked — this is just for labelling your changes
  7. Click "Finish" — GitHub Desktop is ready!
💡 Don't have a GitHub account yet? Go to github.com and click "Sign up" — it's completely free.
Step 2

Create your project repository

A repository is like a folder for your website — GitHub will host it for free

  1. Open GitHub Desktop
  2. Click File → New Repository (or press Ctrl+N / Cmd+N)
  3. Give it a name — use lowercase letters and dashes only (e.g. my-portfolio)
  4. Choose where to save it on your computer (Desktop is fine)
  5. Tick "Initialize this repository with a README"
  6. Click "Create Repository"
  7. Click "Publish repository" in the top bar — make sure it says Public (GitHub Pages only works on public repos for free accounts)
  8. Click "Publish Repository" — you're done!
  1. Open GitHub Desktop
  2. Click File → Clone Repository (or press Ctrl+Shift+O / Cmd+Shift+O)
  3. Switch to the "GitHub.com" tab — your repositories will appear
  4. Click the repository you want to use
  5. Choose where to save it on your computer
  6. Click "Clone"
  7. Your repository is now on your computer and synced with GitHub
💡 Remember your repository name — you'll select it in Ready4Launch in the next steps.
Step 3

Enable GitHub Pages

This is what turns your repository into a live website — and it's free

  1. Go to github.com and sign in
  2. Click on your repository name
  3. Click "Settings" (the gear icon in the top tabs)
  4. Scroll down the left sidebar and click "Pages"
  5. Under "Branch", click the dropdown and select "main"
  6. Leave the folder as "/ (root)"
  7. Click "Save"
  8. Wait about 60 seconds, then refresh — you'll see a green banner with your website's URL!
💡 Your website will be at: yourusername.github.io/your-repo-name — share it with anyone!
